Question-1: What are the different ways to select an element in the DOM ?
There are various ways to select elements in the DOM: by ID (for unique elements), by class name (for elements with the same class), by tag name(for elements of a specific type), by the name attribute (often used in forms), or using query selectors (which allow for more complex, CSS-like queries). Additionally, query selector all can select multiple elements, and traversal methods enable selecting elements based on their relationships (parent, child, or sibling). Each method offers different levels of specificity and flexibility.
Question-2: What is the difference between innerHTML, innerText and TextContent?
`innerHTML` gets or sets the HTML content, including tags. `innerText` retrieves or sets the visible text, considering CSS styling. `textContent` gets or sets all the text inside an element, ignoring HTML tags and styling.
Question-3:What is the event delegation in the DOM?
Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of individual child elements. This parent element listens for events on its children by taking advantage of event bubbling, where events propagate from the target element up to the parent. This allows you to handle events for multiple child elements dynamically, even if new elements are added after the event listener is set, improving performance and reducing memory usage.
Question-4:What is the event bubbiling in the DOM?
Event bubbling is a mechanism in the DOM where an event, triggered on a specific element, propagates upward through its ancestors in the DOM hierarchy. When an event occurs on an element, it first triggers the event listener on that element (the "target"), and then the event bubbles up to its parent, then to the parent's parent, and so on, until it reaches the root of the document (the `document` object). This allows parent elements to listen for and handle events triggered by their child elements, making it possible to use techniques like event delegation.
Question-5:How do you create,add and remove element using Javascript?
In JavaScript, I can create a new element using `document.createElement()`. To add it to the DOM, I use methods like `appendChild()` or `append()`. To remove an element, I can use `removeChild()` on its parent or the `remove()` method directly on the element itself. These methods allow to dynamically modify the DOM by adding or removing elements as needed.
